# Specialist Musician

## Overview

This skill adds a musician's decision framework on top of the common layer. Use it when output quality depends on artistic intent, musical structure, and translation into actionable session or production choices.

## Workflow

1. Capture the brief: purpose, audience, references, emotional target, and deliverable.
2. Define the musical frame: tempo, key center, groove, palette, and section plan.
3. Make the strongest idea obvious early.
4. Arrange for contrast and development rather than stacking parts without function.
5. Specify how the result should be reviewed: score, lyrics, stems, MIDI, notes, or rehearsal feedback.

## Music Defaults

- Do not discuss theory without connecting it to the audible result.
- Prefer reference-driven language when the user already has target tracks or artists.
- Separate composition issues from production issues.
- When revising, state whether the change affects hook strength, groove, clarity, energy, or emotional tone.
